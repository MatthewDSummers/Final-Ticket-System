The Guide

Admin

General

  • The app is designed to meet a company's specific needs
      • {% comment %}
    • Automatic Priorities for tickets can be defined by admins and enabled/disabled
      • {% endcomment %}
    • Custom categories and automations for tickets
    • Custom canned responses
    • Custom dashboard (Staff or Super-Admin)
    • Custom e-mailer (In development. Works in-house)
    • The first user created will be a Super-Admin with all privileges
  • Dark/Light mode

Home

  • Add a frequently visited website by clicking the blue plus icon.
  • Name your resource
  • Paste the full link
  • Remove a resource by clicking the red minus. icon.

Inbox

  • Standard inbox functionality

New

  • Automation - define a new automation for handling tickets (Super-Admin only)
  • Category - define a new category. When a ticket is created, the user selects a category
    ⚬ Super-Admins can delete categories
  • Response - define a new canned response which can be linked with an automation
  • Ticket - create a new ticket
  • User/Admin - create a new user or admin
    ⚬ Only a Super-Admin can make a new Super-Admin

Tickets Table (General)

  • Click any of the table headers to interact and filter the table

Pinned

  • From any page having a table of tickets, any ticket can be pinned
  • The Pinned page is a collection of every non-archived ticket that has been pinned

Active

  • A table of every ticket that is not archived, not on hold, and not resolved

Resolved

  • If a ticket is Resolved and not archived, it will appear here

Withheld

  • If a ticket is placed on Hold, it will appear here

Bots

  • A list of every custom automation (Super-Admin only)
  • (In development. Eventually, you will be able to edit the automation from this page)

Analytics

  • Provide a single date in the top field in order to view tickets from that date
  • Alternatively, provide a date range
  • If there are tickets within the range, you will be presented with many options
  • "See Full Report" will provide a printable version of every ticket within range
    • basically works. If a list is too long, error; margins are also a problem. (In development)
  • Below "See Full Report", you will see methods for filtering the results
  • "Report Type" will present you with pie or line charts

Archives

  • A table of every ticket that is archived
  • If removed, they will return to the Tickets page

Users

  • View all tickets you have created and tickets assigned to you
  • View other people's tickets by seeing their profile
  • All Users - A table of all users (can be reached from any Profile)
    • Select a user to see their profile
    • From there, you can start/revoke their admin status (Super-Admin only)
    • If only one Super-Admin account exists, it can't be deleted,
      and its Super-Admin status can't be revoked.
